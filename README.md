Desktop Reminder Notification System

A Python Tkinter–based desktop application that helps users schedule reminders with desktop notifications and voice alerts.
The system allows users to set reminders with a title, message, date, time, and icon, improving task management and productivity.

Technologies Used

Python

Tkinter – GUI design

Plyer – Desktop notifications

gTTS – Text-to-speech

Pygame – Audio playback

Pillow (PIL) – Image handling

tkcalendar – Date selection
